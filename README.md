# Nimble Gamer Storefront

A high-end, responsive gamer storefront built with React, Vite, and Tailwind CSS.

## Tech Stack
- React + Vite
- Tailwind CSS v3
- Lucide React Icons
- Google Fonts (Inter)

## Development
```bash
npm install
npm run dev
```

## Deployment
Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.
